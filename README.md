# Zygor Guides Viewer (Brasil)

Zygor Guides Viewer e o addon de guias mais completo para World of Warcraft, agora otimizado e traduzido para a comunidade brasileira.

## Destaques
- Guias de Nivelamento: Suba de nivel rapidamente com rotas otimizadas.
- - Guias de Ouro: Melhores formas de farmar ouro no jogo.
  - - Profissoes: Passe de nivel em todas as profissoes de forma facil.
   
    - ## Instalacao
    - Para instalar e manter o seu addon sempre atualizado, recomendamos o uso do WowUp:
    - 1. Baixe o WowUp (https://wowup.io/).
      2. 2. Adicione este repositorio do GitHub no WowUp.
         3. 3. Clique em 'Install' e o addon sera colocado automaticamente na sua pasta do WoW.
